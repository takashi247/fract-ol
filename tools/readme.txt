---

fract-ol: a program to display images of 3 types of fracals (Mandelbrot set, Julia set, and Burning ship)

[Usage]

- Draw mandelbrot set:
  ./fractol m

- Draw julia set:
  ./fractol j [real_part_of_c] [imag_part_of_c]

- Draw burning ship:
  ./fractol b

[Key actions]

- Arrow keys: Shift the displayed area

- C: Toggle color mode

- R/G/B: Change the base-hue to red/green/red respectively

[Mouse actions]

- Mouse wheel: Zoom in/out to the mouse position on the screen
